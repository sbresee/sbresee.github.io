WURFL Capabilities

is_mobile
Virtual capability that is true if a device is wireless, specifically a mobile phone or tablet; alias for is_wireless_device.

can_assign_phone_number
Static capability that is true if a device is a mobile phone and may have a phone number associated to it.

gps
Static capability that returns true if the device has a sensor to determine location using satellites.
